This is a visualizer for one of the key concepts of linear algebra: the transformation. In 2-space, a transformation can be described by a 2x2 matrix. To understand how this matrix will transform our space (or any vector within it), take a look at the red and green 'basis' vectors above. Each one has a name: red => î => say 'i hat' green => ĵ => say 'j hat' You may have done matrix multiplication in school. First row times first column, first row times second column, etc. Here's the intuition. The 1st column of the transformation matrix tells the red î vector where to go. The 2nd column of the transformation matrix tells the green ĵ vector where to go. Now clear whatever you've been futzing around on with the "Restart!" button. Enter the example matrix above and press "Transform!" -1 , 0 , 1 , 1 î ends up at (-1,0) while ĵ ends up at (1,1). Neat, huh? Play around with it for a bit. See if you can define a rotation, a reflection, or a "skew". The nice thing about linear algebra transformations, thus defined, is that they pack a big punch. How nice to be able to describe a specific stretch, turn, and skew with just 4 numbers!
